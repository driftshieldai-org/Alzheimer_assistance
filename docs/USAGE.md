# Usage Guide

This document explains how to interact with the Alzheimer Assistance AI.

---

# Running the Application

Start the assistant:

```
python main.py
```

---

# Example Commands

Reminder creation:

```
Remind me to take medicine at 8 PM
```

Response:

```
Reminder scheduled for 8 PM
```

---

Date query:

```
What day is it today?
```

Response:

```
Today is Monday
```

---

Daily schedule:

```
What do I need to do today?
```

Response:

```
You have 2 reminders today.
```
