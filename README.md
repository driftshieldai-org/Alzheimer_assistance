# Alzheimer Assistance AI

An AI-powered assistant designed to support individuals living with Alzheimer’s disease and their caregivers. The system provides intelligent reminders, conversational guidance, and cognitive support to help patients manage daily tasks and maintain independence.

This project was developed for the **Gemini Live Agent Challenge** to demonstrate how AI agents can provide real-time healthcare assistance.

---

# Features

* Conversational AI assistant
* Context-aware assistance
* Modular architecture
* Extendable for voice or wearable devices

---

# Quick Start

Clone the repository:

```
git clone https://github.com/driftshieldai-org/Alzheimer_assistance.git
cd Alzheimer_assistance
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the assistant:

```
python main.py
```
